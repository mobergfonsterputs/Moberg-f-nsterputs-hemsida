Allmänna villkor - Moberg fönsterputs

Dessa villkor beskriver hur tjänsten fungerar, vad som gäller för pris och betalning samt ansvar för både kund och utförare.

1. Tjänsten

1.1 Vad vi gör

Moberg Fönsterputs erbjuder professionell rengöring av fönster och närliggande ytor. Vi arbetar inom de områden vi aktivt erbjuder tjänster i enligt bokningskalendern och kan vid behov justera vårt arbetsområde eller schema.

1.2 Abonnemang

Vid återkommande fönsterputs sker arbetet enligt ett planerat schema som vi tillsammans kommit överens om.

Om något ändras informerar vi dig i förväg.

1.3 Tidsbokning

Vid bokade tider gäller tiden som riktlinje. Mindre avvikelser kan förekomma.

Om vi blir kraftigt försenade kontaktar vi dig. Kan arbetet inte utföras enligt plan, bokar vi om till en ny tid.

Du ansvarar för att vi får tillgång till fastigheten enligt överenskommelse.

1.4 Kommunikation

Efter utfört arbete får du ett kvitto via SMS eller e-post, undantag kan förekomma.

2. Pris och betalning

2.1 Prissättning

Pris baseras på ett fast pris per fönster. De priser som gäller i prisberäknaren är de priser som gäller.

Det pris du angett vid bokning är inte bindande, det betyder att det kan behöva korrigeras på plats. Vid korrigering uppåt med mer än 20% kontaktas du innan arbetet fortsätter.

2.2 Prisändringar

Vi kan justera våra priser över tid.

2.3 Fakturering

Faktura skickas efter utfört arbete om inget annat avtalats.

2.4 Betalning

Privatpersoner: 15 dagar

Företag: 30 dagar

Vid sen betalning tillkommer lagstadgad påminnelseavgift och ränta.

2.5 RUT-avdrag

Vi hanterar RUT-avdrag direkt med Skatteverket.

Om avdraget inte godkänns är du som kund betalningsskyldig för resterande belopp.

3. Bokning och avbokning

3.1 Avbokning

Du kan avboka utan kostnad fram till en timme innan planerat utförande.

Vid sena avbokningar eller om vi inte får tillgång till fastigheten kan vi debitera för den timmen.

3.2 Återkommande tjänster

Abonnemang kan pausas eller avslutas när som helst.

4. Kundens ansvar

4.1 Tillgång och framkomlighet

Du ansvarar för att:

  • vi kommer åt fönstren utan att behöva stå i sängar eller soffor
  • grindar/larm är öppna

Om vi hindras i vårt arbete kan extra kostnader tillkomma.

4.2 Föremål i vägen

Vi flyttar inte möbler. Krukor, dekorationer eller annan egendom kan vi ta hand om men, om en stor möbel står i vägen ansvarar vi inte för att fönster bakom putsas – och vi tar heller inget ansvar för skador på föremål som är i vägen för arbetet.

5. Begränsningar och ansvar

5.1 Arbetsmiljö

Vi genomför alltid en säkerhetsbedömning. Om arbetet innebär risk kan vi avstå delar av uppdraget.

5.2 Känsliga material

Fönster eller fasader som kräver särskild hantering (t.ex. solfilm, plexiglas eller liknande) ska meddelas i förväg.

Annars ansvarar vi inte för eventuella skador.

5.3 Kraftig smuts

Vanlig fönsterputs omfattar inte borttagning av:

  • färg
  • cement
  • hårt sittande partiklar

Sådant kan kräva specialbehandling.

5.4 Befintliga skador

Om fönster redan är repiga eller skadade ska detta meddelas innan arbetet.

Vi ansvarar inte för befintliga skador.

6. Reklamation - Nöjdhetsgaranti

Om du inte är nöjd vill vi att du hör av dig inom 3 dagar.

Vi åtgärdar felet så snabbt som möjligt.

7. Force majeure

Vi ansvarar inte för förseningar eller uteblivet arbete vid händelser utanför vår kontroll (t.ex. väder, olyckor eller liknande).

8. Personuppgifter

Vi behandlar dina uppgifter för att kunna utföra tjänsten och kommunicera med dig.

Du har rätt att begära utdrag, rättelse eller radering enligt GDPR.

Uppdaterade och gäller från och med 2026.04.09

För uppdraget gäller konsumenttjänstlagen (1985:716) och dessa Allmänna villkor.
